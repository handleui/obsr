## General

### What is Detent?

Detent is an AI-powered CI tool. When your GitHub Actions fail, Detent captures the failure evidence, extracts issue diagnostics, writes fixes using Claude, and lets you apply them with one click.

### How is this different from Copilot?

Copilot helps you write code. Detent fixes code that's already broken — after you push, when CI fails. They complement each other.

### What languages do you support?

Any language where the failure output contains usable evidence. TypeScript, JavaScript, Go, Rust, and Python work best. Structured reports help, but raw logs also work. See [Supported Tools](/docs/cli/tools).

### Do I need to install the CLI?

No. Detent works entirely through GitHub Actions. The CLI is optional — it lets you view errors locally and manage projects.

## Security

### Is my code sent to AI?

Yes, but only the evidence needed to explain and fix the issue. That usually means failure output, workflow context, and the focused code needed for the fix, not your full codebase.

### Where does my code run?

AI resolving runs in isolated sandboxes. Each resolve gets a fresh environment that's destroyed after. Nothing persists between resolves.

### Can I self-host?

Not yet. Contact [sales@detent.sh](mailto:sales@detent.sh) for enterprise options.

### Are you SOC 2 compliant?

Yes, SOC 2 Type II. All data is encrypted at rest and in transit.

## Pricing

### How much does it cost?

You pay per successful resolve. Autofix (formatting, imports) is free. AI resolves cost 1-5 credits depending on complexity. See [Pricing](/docs/pricing).

### Is there a free tier?

Yes. New accounts get 50 free credits — enough for ~30-50 resolves.

### What if the fix doesn't work?

You're only charged for successful resolves. If a fix fails or you reject it, no charge.

## Troubleshooting

### Why aren't my errors being detected?

Make sure the failing command output is available to the action. Detent can work from raw console output, but better file paths, line numbers, stack traces, and command names improve results. See [Supported Tools](/docs/cli/tools) for guidance.

### Why did my resolve fail?

Complex logic bugs, missing dependencies, or architectural issues may be beyond what AI can fix. Detent flags these for human review instead of guessing.

### The action isn't running

Check that you're using `if: always()`. Without this, the action only runs when previous steps succeed — which defeats the purpose.
