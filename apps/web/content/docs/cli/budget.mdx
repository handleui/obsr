## How It Works

Detent charges per successful heal. You buy credits, and each heal uses some.

## Credit Usage

| Heal Type | Credits |
|-----------|---------|
| Autofix (deterministic) | Free |
| AI Heal (simple fix) | 1 credit |
| AI Heal (complex fix) | 2-5 credits |

Autofix handles formatting, import sorting, and other deterministic fixes. No cost.

AI heals use Claude to understand and fix your code. Cost scales with complexity.

## Set a Budget

In [Navigator](https://navigator.detent.sh), you can set a monthly budget per project. When you hit the limit, heals pause until next month (or you add more credits).

## View Usage

Check your current usage:

1. Go to [Navigator](https://navigator.detent.sh)
2. Click your project
3. See "Usage" in the sidebar

Or via API:

```bash
curl -H "Authorization: Bearer $DETENT_TOKEN" \
  https://backend.detent.sh/v1/usage
```

## Add Credits

1. Go to [Navigator](https://navigator.detent.sh)
2. Click "Add Credits"
3. Pay with card

Credits don't expire.

## Free Tier

New accounts get 50 free credits. Enough for ~30-50 heals.
